[b]Features[/b]

Adds a new custom button with a label and URL of your choosing, before the Home button.

[hr]
[b]Configuration[/b]
Website configuration settings are located in:
Admin -> Configuration -> Modifications -> Website Button
The following settings are provided:

Activate Website Button: Activates the website button.
Website tab label: Enter your label text here. Defaults to Website.
Website tab execute URL: Enter your URL here. Defaults to your forum's main index URL.
