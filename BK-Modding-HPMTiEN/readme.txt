[b]Hide PM Text in Email Notifications by BK Modding[/b]

Version: 1.0 | Date: July 23, 2013
Compatibility: SMF 2.0

Made by: [url=http://www.simplemachines.org/community/index.php?action=profile;u=194815]BK Modding Team[/url] - [url=http://www.bkmodding.com/]BK Modding[/url]

This is a simple mod that hides the pm text in email notifications.
This was requested on the [url=http://www.simplemachines.org/community/index.php?topic=501395.0]SMF site[/url] by a member, and was originally posted by: [url=http://www.simplemachines.org/community/index.php?action=profile;u=93211]shadow82x[/url]

There are no settings, so if you want to disable this mod, just uninstall it via the package manager.
