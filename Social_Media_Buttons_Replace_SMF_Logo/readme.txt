[b]Social Media Buttons[/b]

Version: 1.0 Date: December 31, 2014
Compatibility: [b]SMF 2.0.x[/b]

Made by: [url=http://www.burkeknight.com/index.php?action=profile;u=1]BurkeKnight[/url] - [url=http://www.burkeknight.com/]BurkeKnight Enterprises[/url]

This modification adds a way for the admin to add Social Media Buttons to their site, replacing the SMF logo.

This mod is made for the default theme. If you want it into a custom theme, you will have to manually edit it.

SMF 2.0.x: Admin > Current Theme
