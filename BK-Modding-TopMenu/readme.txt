Compatibility: [b]SMF 2.0.x[/b]

This was originally a [url=http://www.simplemachines.org/community/index.php?topic=444219.msg3123598#msg3123598]post[/url] by: [url=http://www.simplemachines.org/community/index.php?action=profile;u=129726]Antechinus[/url]

[b]NOTES: This mod is for Curve (Default) theme and may well not work on custom themes and will require per-theme changes for them.
Since this mod does css changes, you will need to clear your browser cache or do a hard refresh after install.[/b]
